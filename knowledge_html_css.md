# Language Knowledge File: HTML & CSS

## 📄 HTML Fundamentals
**Purpose:** Defines the structure and content of a web page (the skeleton).
**Key Concepts Learned:**
*   **Structure:** Understanding the basic document structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`).
*   **Elements & Tags:** Using semantic elements like `<header>`, `<nav>`, `<article>`, `<h1>` through `<h6>`, `<p>`, `<a>`, and `<img>`.
*   **Attributes:** How attributes modify elements (e.g., `href` for links, `src` for images).

## 🎨 CSS Fundamentals
**Purpose:** Controls the presentation and visual styling of HTML content (the look and feel).
**Key Concepts Learned:**
*   **Selectors:** Techniques for targeting specific HTML elements (class selectors `.`, ID selectors `#`, element selectors).
*   **The Box Model:** Understanding the four components: Content, Padding, Border, and Margin.
*   **Layout:** Familiarity with modern layout methods such as **Flexbox** (for one-dimensional layouts) and **CSS Grid** (for two-dimensional layouts).
