# Language Knowledge File: PHP

## 🐘 PHP Fundamentals
**Purpose:** A server-side scripting language primarily used for web development to handle dynamic content, interact with databases, and manage application logic.
**Key Concepts Learned:**
*   **Server-Side Execution:** Understanding that PHP code runs on the web server, not in the user's browser (unlike JavaScript).
*   **Syntax:** Basic syntax, variables, control structures (if/else, loops), functions, and arrays.
*   **Web Interaction:** How PHP interacts with HTTP requests (GET/POST methods) to handle form submissions and data exchange.
*   **Database Integration:** Concepts around connecting to databases (like MySQL) using PDO or MySQLi extensions to perform CRUD operations (Create, Read, Update, Delete).

## ⚙️ Object-Oriented Programming (OOP) in PHP
**Key Concepts Learned:**
*   **Classes and Objects:** Defining classes, creating objects, properties, and methods.
*   **Inheritance & Encapsulation:** Implementing OOP principles to create reusable and maintainable code structures.
